# Sketch 001: Direction Comparison

## Design Question

Two complete visual directions for the Caleb Lim portfolio, each shown across two screens (splash + Graphic Design category). Same content, same UX flow, opposite design languages. Pick one — that becomes the design system anchor for all subsequent phases.

## How to View

```
open .planning/sketches/001-direction-comparison/index.html
```

Use the floating black pill at the top to swap between A and B. Both directions show the splash and the Graphic Design category page stacked — scroll within each variant to see both.

## Variants

- **A — Editorial brutalist.** Off-white paper, near-black ink, single ink-red accent. Big *Fraunces* serif (variable, with `WONK` and italic axis used). Asymmetric editorial grid with broken column alignment. Mono-font micro-labels. Reads as "analyst who can also design" — quiet confidence, premium type.

- **B — Magazine maximalist.** Warm cream + dark ink + four saturated brand colors (terracotta / cobalt / electric lime / plum). *Bricolage Grotesque* set huge and tight, mixed with italic *Fraunces* for accent. Cards rotated, layered, decorative; pulsing "open to roles" status pill. Reads as "brand creative with finance chops" — confident and loud.

## What to Look For

- **Which one would a recruiter screenshot and Slack to a hiring manager?** That's the test.
- **How does each handle the asymmetric piece counts?** Finance has 3 pieces; Personal has 2. Direction A's editorial grid breathes with low piece counts; Direction B's color-block layout makes a 2-piece gallery feel intentional, not empty. Both addressed but differently.
- **Type pairings.** A leans on a single-family workhorse (Fraunces in display + italic + mono). B mixes Bricolage (grotesque sans) and Fraunces italic for editorial accent.
- **Color load.** A has one accent color used sparingly. B uses ~five hues actively. Either works for a brand portfolio — they signal different things about you.
- **The "What do you wish to see?" prompt.** A treats it as quiet handwritten editorial copy. B treats it as a typographic shout with a "→ PICK ONE" marker tag.
- **Anti-AI tells.** Neither uses centered hero, gradients, shadcn cards, lucide icons, or Inter. Direction A actively rejects color; Direction B uses color the way Readymag templates do (large flat blocks, not gradients).
